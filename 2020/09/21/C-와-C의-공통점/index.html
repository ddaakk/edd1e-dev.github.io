C++ 와 C 의 공통점

Author: Kang Sang Woo Date: September 21, 2020  12:30:54 Category: C++

변수

변수의 정의

사실 C 언어에서 작성된 코드를 그대로 C++ 에 붙여 넣기 해도 큰 문제가 없다고 말해도 과언이 아닐 정도로 C++ 은 C 언어의 문법을 거의 완전하게 포함하고 있습니다.

가장 기초적인 변수를 정의하는 부분입니다. 위 코드를 볼 때 C 언어에서 작성한 코드라고 말해도 이상하지 않습니다. 변수를 정의하는 부분에서 만큼은 C 언어와 달라진 점이 없습니다.

변수 명 이름 작성 규칙도 바뀐 것이 없습니다. 변수명도 C 언어 때와 마찬가지로 알파벳과 _ 기호, 숫자들을 사용할 수 있으며 그 외 것들은 사용할 수 없습니다.
또한 변수 이름의 맨 앞 부분에는 숫자가 올 수 없습니다.

Google의 C++ 변수의 이름을 짓는 규칙인 C++ Coding Standard는 이곳에서 확인하실 수 있습니다.

변수를 짓는데 모든 공통된 사항으로는 변수의 이름만 보았을 때 직관적으로 무슨 역할을 하는 것인지 알 수 있어야 합니다.
