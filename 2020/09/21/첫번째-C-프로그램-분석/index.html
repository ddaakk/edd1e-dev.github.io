이름 공간(namespace)

먼저 cout 앞에 붙어 있는 std 에 대해 알아보겠습니다. std 는 C++ 표준 라이브러리 의 모든 함수, 객체 등이 정의된 이름 공간(namespace) 입니다.

이름 공간이란 말그대로 어떤 정의된 객체에 대해 어디 소속인지 지정해주는 것 입니다.

코드의 크기가 늘어나거나 다른 사람들이 쓴 코드를 가져다 쓰게 되면서 중복된 이름을 가진 함수들이 많아졌습니다. 따라서 C++ 에서는 이를 구분하기 위해, 같은 이름이라도 소속된 이름 공간(namespace) 이 다르면 다른 함수로 취급하게 되었습니다.

위의 경우 std 라는 이름 공간에 정의되어 있는 cout 함수를 의미합니다. 만약에 std:: 없이 그냥 cout 이라고만 코드를 작성한다면 컴파일러가 cout 을 찾지 못합니다.
